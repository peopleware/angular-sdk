# navigation models

## What They Are For

Use the navigation models to describe the shell structure:

-   `NavigationItem`
-   `SidebarOptions`

## Usage Guidelines

-   Use `NavigationItem` for links, groups, external navigation entries, and sidebar hierarchy.
-   Use `SidebarOptions` for shell-level display behavior such as logo or default open and close behavior.
-   Keep navigation configuration declarative so the shell can remain reusable.
