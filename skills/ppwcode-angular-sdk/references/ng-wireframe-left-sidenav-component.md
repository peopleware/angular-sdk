# `LeftSidenavComponent`

## When To Use

Use `LeftSidenavComponent` directly when a feature needs the SDK navigation panel without the full wireframe shell.

## Usage Guidelines

-   Prefer using it through `WireframeComponent` first.
-   Pass well-structured `NavigationItem` data rather than hardcoding menu rendering around the component.
-   Keep custom styling aligned with the package’s CSS variable theming model.
