# `DraggableDialogDirective`

## When To Use

Use `DraggableDialogDirective` on Angular Material dialogs when users benefit from moving the dialog out of the way while still keeping it open.

## Usage Guidelines

-   Apply it to dialogs that can block underlying context the user still needs to see.
-   Pair it with a clear drag handle so discoverability stays reasonable.
-   Keep it as an enhancement, not a requirement for basic task completion.
