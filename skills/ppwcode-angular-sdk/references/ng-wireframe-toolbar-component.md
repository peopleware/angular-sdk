# `ToolbarComponent`

## When To Use

Use `ToolbarComponent` when a screen or shell needs the SDK toolbar behavior without necessarily adopting the entire wireframe.

## Usage Guidelines

-   Use it directly only when the full `WireframeComponent` would be too much.
-   Prefer projecting custom content into the wireframe toolbar area when the page already uses `WireframeComponent`.
-   Keep toolbar actions high-signal and sparse.
