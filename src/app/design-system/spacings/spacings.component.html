Spacings define distance inside of between elements.

Examples

@for (spacing of spacings; track spacing) {
}

CSS usage

    .my-element {
        display: flex;
        flex-direction: row;
        gap: var(--pds-spacing-4);
    }

HTML usage

    <div class="pds-gap-4">...</div>
    <div class="pds-ms-4">...</div>
    <div class="pds-mx-4">...</div>
    <div class="pds-py-4">...</div>
