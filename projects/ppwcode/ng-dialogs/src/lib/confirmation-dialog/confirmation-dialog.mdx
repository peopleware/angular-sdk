# Confirmation Dialog

The `ConfirmationDialogComponent` provides a flexible and reusable dialog for standard "confirm/cancel" workflows. It is built on top of Angular Material's `MatDialog`.

## When to use
- To confirm a destructive action (e.g., deleting a record).
- To notify the user about an important event that requires acknowledgement.
- To ask a simple "Yes/No" or "Confirm/Cancel" question.

## Basic Usage

The dialog is typically opened using `MatDialog.open` with `ConfirmationDialogComponent` and the necessary data.
