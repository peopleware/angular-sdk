# Peopleware Angular SDK Storybook

This Storybook showcases the components and features of the Peopleware Angular SDK. The SDK is a collection of libraries designed to accelerate Angular development with reusable, accessible, and well-tested components.

## Libraries with Storybook stories

The following libraries currently have stories available in this Storybook:

### 📦 ng-common-components
A collection of reusable UI components for data display and user interaction.
- **Table**: A flexible data table with support for custom column types, sorting, row selection, and expandable details.
- **Expandable Card**: A collapsible card with customizable headers and state-driven indicators.
- **Message Bar**: Severity-based informational banners with customizable styling.
- **Search Filter**: A standardized layout for building search forms and filtering results.
- **Loader**: A simple yet customizable loading spinner for asynchronous operations.
- **Dashboard Items Table**: A responsive grid layout designed specifically for hosting dashboard summary items.

### 💬 ng-dialogs
Utilities and reusable dialogs to simplify common user interaction patterns.
- **Confirmation Dialog**: A flexible dialog for standard "confirm/cancel" workflows.
- **Draggable Dialogs**: A directive to easily make standard Material dialogs interactive and movable.

### 🌐 ng-router
Provides components and services for application routing and navigation.
- **Breadcrumb**: Indicates the current location in the application.
- **Pagination**: Utilities for handling paged data in the URL.
- **Relative Navigation**: Mixins for relative navigation handling.

### 🏗️ ng-wireframe
Provides a structural framework for Angular applications.
- **Wireframe**: A complete application layout with responsive navigation.
- **Left Sidenav**: A collapsible navigation menu with support for nested items and logos.
- **Toolbar**: A top bar for actions, search, and page titles.
- **Pagination Bar**: A wrapper around Angular Material's paginator for consistent styling.

For more details on each library, please refer to their respective `README.md` files in the `projects/ppwcode/` directory.
